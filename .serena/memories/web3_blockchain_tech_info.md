# Web3・ブロックチェーン技術情報

## サポートするブロックチェーン

### Ethereum
- **開発言語**: Solidity
- **フレームワーク**: Hardhat
- **ライブラリ**: ethers.js, viem, wagmi
- **ウォレット**: MetaMask

### Polygon
- **特徴**: EVM互換、ガス効率重視
- **開発環境**: Ethereumと同様

### Avalanche
- **特徴**: C-Chain、Subnet作成
- **開発環境**: EVM互換

### NEAR
- **開発言語**: Rust、AssemblyScript
- **ウォレット**: NEAR Wallet

### Solana
- **開発言語**: Rust
- **フレームワーク**: Anchor Framework
- **ウォレット**: Phantom

### ICP (Internet Computer)
- **開発言語**: Motoko
- **特徴**: Internet Computer Protocol

### Astar
- **特徴**: Polkadot Parachain
- **開発環境**: EVM互換 + Wasm

### XRPL (XRP Ledger)
- **特徴**: XRP Ledger専用機能
